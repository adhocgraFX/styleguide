Typographie

Die Proportionen der Absatz- und Überschriftenstile von Jflex beruhen auf einer harmonischen und modularen typografischen Tonleiter, die sich den unterschiedlichen device Größen dynamisch anpassen. Alle Absätze, Überschriften und deren Zeilenhöhen sind proportional, und nicht mit festen Pixelwerten definiert. So bleibt in der Skalierung das ursprüngliche Aussehen des layouts erhalten.

Bilder werden proportional skaliert, aber nicht vergrößert.

Video streams werden ebenfalls proportional skaliert.

  2. max-image = 100%, kann vergrößert werden
  3. two-thirds-image = 66%
  4. half-image = 50%
  5. third-image = 33%

Module Klassen Suffixe

Folgende module class suffixes stehen zur Verfügung:

1. Farbklassen
  • white
  • clouds = hellblaugrau
  • concrete = zementgrau
  • midnightblue
  • jugfuldablue
  • turquoise
  • blue, info
  • my-red, alert
  • green, success
  • yellow, question,
  • orange
  • transparent
2. Klassen für menüs in der sidebar:
  • style
  • menu-style
3. Klasse für das Modul Suche:
  • search-style
4. Klassen für die Modul-Breiten in %:
  • grid-5 bis grid-100; in 5er Schritten
  • grid-33, grid-66 = 1/3 und 2/3
  • grid 62, grid 38 = goldener Schnitt
5. Helfer-Klassen:
  • hide-on-mobile, hide-on-desktop
  • equal-1, equal-2: Module in einer Reihe können mit equal-1 bzw. equal-2 auf gleiche Höhe gebracht werden.
  • stickem: Das Modul bleibt beim "scrollen" unterhalb der Suche "kleben" (nicht im mobile Modus).
